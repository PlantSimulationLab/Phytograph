# User Guide

Welcome. This guide walks you through Phytograph from a fresh install to
analyzing your own plant scans.

If you've never used the app before, read these three pages in order:

1. **[Install Phytograph](install.md)** — download and open the app on macOS or Windows.
2. **[Your first import](first-import.md)** — load an example dataset and see your scan in 3D.
3. **[Tour of the interface](interface-tour.md)** — what every panel, toolbar, and color-coded button does.

Once you're comfortable with the interface, jump to the **[Workflows](../workflows/index.md)** section for step-by-step recipes for specific tasks (clean a noisy scan, triangulate a mesh, generate a plant, etc.).

If you want to understand the **data types** Phytograph works with —
point clouds, meshes, skeletons, plant models, scans — read the
**[Concepts](../concepts/index.md)** section. You can come back to it
any time; the workflows link out to the relevant concept when needed.
